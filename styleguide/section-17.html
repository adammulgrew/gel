17 Typography

17.1 Standard Life Font

We have our own font called Standard Life font.

It is unique to Standard Life.

It is the only font that should be used for headlines, titles, headers, sub heads.

It can also work well for short body copy, especially at larger sizes.

To use Standard Life add class="primary-font" or class="primary-font-bold" to the required tag or parent element or include font-family:$primary-font or font-family:$primary-font-bold" in the css class

Regular
Aa1
Bold

17.2 Meta Standard Life Pro TF

For general body copy, and tables and charts, we use our own version of Meta called “Meta Standard Life Pro TF”.

To use Meta Standard Life add class="secondary-font" or class="secondary-font-bold" to the required tag or parent element or include font-family:$secondary-font or font-family:$secondary-font-bold" in the css class

A modular scale

The term Modular Scale refers to a series of harmonious values. As part of the patterns library _typography.scss has a default responsive modular scale tied to the classes below. Note, this is for illutrative purposes only. Each project and a suitable modular scale should be just on an individual basis.

17.5 Color recommendations

Text should only appear at 100% colour on all backgrounds

Text in 100% white
Text in 100% bright-blue
Text in 100% yellow
Text in 100% dark-blue
Text in 100% bright-blue
Text in 100% grey
Avoid using yellow text on white backgrounds
Avoid using orange text on white backgrounds
Avoid using orange text on blue backgrounds
